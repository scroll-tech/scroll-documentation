Kunjungi [Bridge](https://sepolia.scroll.io/bridge) kami untuk memulai! Aplikasi Bridge mendukung baik operasi **Deposit** maupun **Withdraw**, memungkinkan pengguna untuk memindahkan aset dari Sepolia Testnet ke Scroll Sepolia Testnet tanpa perlu kepercayaan.

Deposit dapat memakan waktu hingga 15 menit untuk tersedia di Scroll.

Withdrawal, yang memerlukan interaksi kedua di Sepolia setelah penarikan selesai, bisa memakan waktu lebih lama.

:::caution[Menunggu lama untuk transaksi bridge?]
Estimasi waktu di atas adalah tipikal untuk perilaku jaringan dan tingkat aktivitas normal. Karena kami hanya memproses sejumlah tertentu transaksi L1 yang di-queue per blok L2, pesan yang di-bridge mungkin memerlukan waktu lebih lama untuk dimasukkan ke Scroll pada saat penggunaan jaringan yang luar biasa tinggi.
:::

## Deposit dari Sepolia ke Scroll Sepolia

### Instruksi

1. Pertama, navigasikan ke [Scroll Bridge](https://sepolia.scroll.io/bridge) dan tekan tombol "Connect Wallet".
2. Di aplikasi, pastikan **Ethereum Sepolia** berada di atas dan **Scroll Sepolia** berada di bawah. Anda dapat mengklik tombol "**↓**" untuk menukar posisi mereka.
3. Pilih token yang ingin Anda transfer dari Sepolia ke Scroll Sepolia. Jika ini pertama kali Anda bridging, coba "ETH".
4. Jika ini pertama kali Anda mentransfer token ERC20 tertentu, Anda harus **Menyetujui** kontrak Sepolia Bridge untuk mengakses token ERC20 Anda.
5. Selanjutnya, klik tombol **Kirim ke Scroll Sepolia** untuk melakukan deposit. Dompet Anda akan meminta konfirmasi transaksi transfer.
6. Setelah transaksi transfer dikirim dan dikonfirmasi, token akan dikurangkan dari dompet Sepolia Anda.
7. Anda selalu dapat memeriksa status transaksi dengan menekan ikon "History" di sebelah alamat dompet Anda di pojok kanan atas.

### Kapan token akan tiba di dompet Scroll Sepolia Anda?

Ini bisa memakan waktu antara **8 hingga 14 menit** (menunggu blok menjadi [_Safe_](https://www.alchemy.com/overviews/ethereum-commitment-levels#what-are-ethereum-commitment-levels) di Sepolia) sebelum token muncul di dompet Scroll Sepolia Anda.

## Tarik dari Scroll Sepolia ke Sepolia

### Instruksi

#### Mengirimkan Transaksi Penarikan Awal Anda

1. Pertama, beralih ke jaringan **Scroll Sepolia** di dompet Anda.
2. Di aplikasi, pastikan **Scroll Sepolia** berada di atas dan **Ethereum Sepolia** berada di bawah. Anda dapat mengklik tombol "**↓**" untuk menukar posisi mereka.
3. Pilih token yang ingin Anda transfer dari **Scroll Sepolia** ke **Sepolia**. Jika ini pertama kali Anda bridging, coba "ETH".
4. Jika ini pertama kali Anda mentransfer token ERC20 tertentu, Anda harus **Menyetujui** kontrak Scroll Sepolia Bridge untuk mengakses token ERC20 Anda.
5. Selanjutnya, klik tombol **Kirim ke Ethereum Sepolia** untuk melakukan penarikan. Dompet Anda akan meminta konfirmasi transaksi transfer.
6. Setelah transaksi transfer dikirim dan dikonfirmasi, token akan dikurangkan dari dompet Scroll Sepolia Anda.

#### Mengirimkan Transaksi Penarikan Execute

_Langkah-langkah yang tersisa terjadi di Scroll Sepolia, tetapi Anda harus menunggu transaksi Anda untuk sepenuhnya terbukti ("finalized") di Ethereum Sepolia. Proses ini bisa memakan waktu hingga empat jam._

1. Ketika transaksi penarikan Anda selesai difinalisasi di Ethereum Sepolia, Anda akan melihat tombol "Claim" di area Transaksi Terbaru menjadi solid.
2. Klik tombol "Claim" untuk mengirimkan transaksi Penarikan Execute.
3. Setelah dikirim, dana yang ditarik harus muncul langsung di dompet Sepolia Anda.

### Kapan token akan tiba di dompet Sepolia Anda?

Token yang ditransfer akan tiba di dompet Sepolia Anda segera setelah blok yang berisi transaksi Penarikan Execute Anda dikonfirmasi di Sepolia.
