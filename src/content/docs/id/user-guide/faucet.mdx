Untuk berinteraksi dengan testnet kami, pertama-tama Anda perlu menerima testnet ETH di _Sepolia Testnet_. Kemudian Anda dapat melakukan bridging dari _Sepolia Testnet_ ke _Scroll Sepolia Testnet_.

## Faucet Sepolia

Setiap kran memiliki aturan dan persyaratan tersendiri, jadi Anda mungkin perlu mencoba beberapa sebelum menemukan yang sesuai untuk Anda.

Berikut adalah beberapa aplikasi kran Sepolia:

- [https://sepoliafaucet.com](https://sepoliafaucet.com/)
- [https://sepolia-faucet.pk910.de](https://sepolia-faucet.pk910.de)
- [https://faucet.quicknode.com/drip](https://faucet.quicknode.com/drip)
- [https://faucet.chainstack.com](https://faucet.chainstack.com)
- [https://infura.io/faucet/sepolia](https://infura.io/faucet/sepolia)

Setelah Anda menerima ETH di Sepolia, Anda seharusnya melihatnya di dompet Anda di _Jaringan Sepolia_. Mungkin memerlukan beberapa detik untuk muncul, tetapi Anda dapat memeriksa statusnya dengan mencari transaksi ke alamat Anda di [Explorer Sepolia](https://sepolia.etherscan.io/).

:::note[Pace Yourself!]
Sebagian besar faucet hanya mengizinkan permintaan token uji sekali setiap 24 jam.
:::
