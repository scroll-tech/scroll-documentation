Terima kasih telah mencoba Testnet Scroll Sepolia. Jika Anda memiliki pertanyaan atau ingin memberikan umpan balik, bergabunglah dengan [Discord](https://discord.gg/scroll) kami!

Testnet Sepolia terdiri dari _Testnet Sepolia Ethereum_ dan jaringan uji _Scroll Sepolia_. Sepolia adalah jaringan uji Ethereum, sementara Scroll Sepolia adalah testnet zero knowledge rollup yang diterapkan di atasnya. Ada beberapa aplikasi demo yang sudah dipasang: sebuah [bridge](https://sepolia.scroll.io/bridge) antara _Sepolia_ dan _Scroll Sepolia_,[^1] sebuah [penjelajah blok](https://sepolia-blockscout.scroll.io/) untuk _Scroll Sepolia_,[^2] dan sebuah [penjelajah rollup](https://sepolia.scroll.io/rollupscan).

Untuk melihat transaksi L1, kunjungi [penjelajah Sepolia](https://sepolia.etherscan.io/) dari Etherscan.
Untuk melihat transaksi L2, Anda dapat menggunakan [Scrollscan](https://sepolia.scrollscan.com) atau [Blockscout Scroll](https://sepolia-blockscout.scroll.io/), tetapi Anda mungkin juga ingin mencoba fungsionalitas tambahan yang disediakan oleh [Dora](https://www.ondora.xyz/network/scroll-sepolia/interactions) atau [L2Scan](https://scroll-sepolia.l2scan.co/).

Berikut adalah alur kerja yang disarankan untuk menjelajahi Testnet:

1. Tambahkan konfigurasi [Sepolia Testnet](https://sepolia.scroll.io/portal) ke dompet Anda.
2. Mintalah token uji di jaringan _Sepolia_ dari aplikasi Faucet Ethereum apa pun. (lihat artikel [Faucet](/id/user-guide/faucet))
3. Transfer token uji dari _Sepolia_ ke _Scroll Sepolia_ melalui aplikasi [Bridge](https://sepolia.scroll.io/bridge).
4. Transfer token ke dompet lain di _Scroll Sepolia_ menggunakan dompet Anda.
5. Jelajahi ekosistem kami, berinteraksi dengan kontrak seperti [Uniswap](https://uniswap-showcase.sepolia.scroll.xyz/) atau [Aave](https://app.aave.com).
6. Tarik token dari _Scroll Sepolia_ ke _Sepolia_ melalui aplikasi [Bridge](https://sepolia.scroll.io/bridge).

Anda dapat menemukan instruksi untuk setiap aplikasi di bagian lain dari panduan pengguna ini.

## Pertanyaan & Umpan Balik
