Grazie per aver testato il Testnet di Scroll Sepolia. Se hai domande o vuoi dare feedback, unisciti al nostro [Discord](https://discord.gg/scroll)!

Il Testnet Sepolia √® composto dal _Testnet Sepolia_ di Ethereum e dalla rete di test _Scroll Sepolia_. Sepolia √® una rete di test di Ethereum, mentre Scroll Sepolia √® una rete di test di rollup a conoscenza zero implementata sopra la prima. Ci sono alcune applicazioni demo pre-implementate: un [bridge](https://sepolia.scroll.io/bridge) tra _Sepolia_ e _Scroll Sepolia_,[^1] un [esploratore di blocchi](https://sepolia.scrollscan.com) per _Scroll Sepolia_, e un [esploratore di rollup](https://sepolia.scroll.io/rollupscan).

Per visualizzare le transazioni L1, dai un'occhiata all'[esploratore di Sepolia](https://sepolia.etherscan.io/) di Etherscan.
Per visualizzare le transazioni L2, puoi utilizzare [Scrollscan](https://sepolia.scrollscan.com), o potresti voler provare anche le funzionalit√† aggiuntive fornite da [Dora](https://www.ondora.xyz/network/scroll-sepolia/interactions) o [L2Scan](https://scroll-sepolia.l2scan.co/).

Ecco il workflow suggerito per esplorare il Testnet:

1. Aggiungi le configurazioni del [Testnet Sepolia](https://sepolia.scroll.io/portal) al tuo wallet.
2. Richiedi token di test nella rete _Sepolia_ da qualsiasi app Faucet Ethereum. (vedi articolo [Faucet](/it/user-guide/faucet))
3. Trasferisci token di test da _Sepolia_ a _Scroll Sepolia_ tramite l'app [Bridge](https://sepolia.scroll.io/bridge).
4. Trasferisci token ad altri wallet su _Scroll Sepolia_ utilizzando il tuo portafoglio.
5. Esplora il nostro ecosistema, interagendo con contratti come [Uniswap](https://uniswap-showcase.sepolia.scroll.xyz/) o [Aave](https://app.aave.com).
6. Ritira token da _Scroll Sepolia_ a _Sepolia_ tramite l'app [Bridge](https://sepolia.scroll.io/bridge).

Puoi trovare le istruzioni per ciascuna app nel resto di questa guida utente.
